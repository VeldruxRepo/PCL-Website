# PCL React Figma Build

React + TypeScript + Vite implementation based on the supplied desktop and mobile Figma exports.

## Run

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run build
```

## Content

Edit `src/data/site-content.json` to change repeated copy, navigation, services, process steps, case studies, experiments, testimonials, metrics, FAQs, and footer links.

## Images

Every intended image location uses the reusable `ImagePlaceholder` component and displays `PLACE IMAGE HERE`. Replace each instance with your final image component or extend the JSON with image URLs later.
